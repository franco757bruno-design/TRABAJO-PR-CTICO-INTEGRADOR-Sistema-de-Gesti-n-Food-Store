# TRABAJO-PR-CTICO-INTEGRADOR-Sistema-de-Gesti-n-Food-Store
Trabajo Práctico Integrador para la materia Programación 2 de la Universidad Tecnológica Nacional (UTN).

Aplicación de consola desarrollada en Java para la gestión de productos, categorías, usuarios y pedidos. El sistema aplica principios de Programación Orientada a Objetos (POO) y gestiona la persistencia de datos en memoria mediante el uso de Colecciones.

## Requisitos del Sistema
* Lenguaje: Java 21
* IDE: NetBeans, IntelliJ IDEA o Eclipse
